üìå Pin Interaction Flow

  1. User sees circular markers with white flat icons on map
  2. User taps on a marker
  3. Marker disappears and custom pin appears at same location
  4. Pin shows same color and white flat icon as the original marker
  5. Tapping elsewhere or another marker removes the pin
  6. Only one pin visible at a time
